At the conclusion of this course, you should be able to:

  • Define what a learning object is
  • Define what the learning object repository is
  • Have an understanding of the LOR is situated within DTCC and your courses
  • Access the LOR directly
  • Perform a retrieval of a learning object using a direct link, locked link, or copy as content method
  • Retrieve content from within course content, course admin, course builder, and course files
  • Publish a learning object to the LOR from course content or course files
  • Update an already published learning object by overwriting it
  • Edit the properties and metadata for a published learning object
  • Delete a published learning object
