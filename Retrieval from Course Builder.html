The Course Builder is a useful tool when in the beginning stages of creating your course. It can be extremely beneficial to import learning objects during this step. For example, when copying entire modules or multiple learning objects it is easier when you are viewing the course in this fashion. View the video above for a demonstration of how to complete this task. Below you will find step-by-step instructions accompanied by screenshots.

Step 1

From your Dashboard, click on a course.

Step 2

From the Navbar click "Admin"

Step 3

Click on "Course Builder"

Note: Before you can add a learning object from the LOR you need to have a Module for it to put into first

Step 4

First Method:

Under “Add Content” (A) on the left, click the “Search for Learning Object” (B) button.

Step 5

Select the module the learning object will be put into

Advance to Step 7

Step 6

Second Method

Drag the “Search for Learning Object” button into the module of your choice

Step 7

Search (A) or browse (B) for a learning object

Step 8

In the list of results select the learning object (A), then click the "Select" button (B)

The object has been added to the course.
